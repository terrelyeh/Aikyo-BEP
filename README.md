# Aikyo 冰店損益平衡分析

愛嬌冰店互動式 BEP（Break-Even Point）試算工具，適合與團隊溝通討論使用。

**Demo →** [Cloudflare Pages 部署連結]

---

## 這是什麼？

一個**互動式損益平衡試算器**，專為愛嬌冰店的經營團隊設計。

這個工具幫助團隊掌握兩件最基本的事：

1. **固定成本到底是多少** — 把房租、人事、折舊等項目攤開來看，知道每個月還沒開門就要花多少錢
2. **損益平衡點在哪裡** — 每個月至少要做到多少營業額，才能剛好打平這些固定成本

有了這個 baseline，團隊就能以此為基礎去規劃盈利策略：要賣什麼組合、定價多少、人力怎麼排、是否需要額外通路來分攤成本。

畫面左側調整參數，右側即時更新結果。純 HTML 單檔，不需要安裝任何東西，用瀏覽器打開就能用。

---

## 解決什麼問題？

經營一間冰店，團隊需要先回答一個根本問題：**「每個月至少要賣多少才不會虧錢？」**

這個工具提供一個可互動的試算環境，讓團隊可以：

1. **看清固定成本結構**：房租、正職薪資、PT 排班、折舊、水電 — 各佔多少比例、哪一項最重，一目瞭然
2. **審視計時人員的人力成本**：調整 PT cover 比例、時薪、人數，即時看到排班成本和勞保勞退的變化，評估不同排班方案對固定成本的影響
3. **找到損益平衡點**：根據產品售價與毛利率，算出打平所需的月營業額，以及對應的每日 / 每小時銷量目標
4. **比較不同產品組合**：全賣霜淇淋？全賣聖代？各半？自訂比例？不同組合的 BEP 差多少，直接對照
5. **模擬酒場合作通路（選用）**：開啟「酒場合作」開關，輸入 B2B 預估月售量，計算返利 20% 後酒場通路能為直營門市分擔多少固定成本
6. **降低溝通門檻**：用視覺化圖表讓非財務背景的夥伴也能直觀理解數字，讓討論有共同的數據基礎

## 如何使用？

### 開啟

直接用瀏覽器開啟 `index.html`，輸入密碼後即可進入主畫面。

```bash
open index.html
```

### 操作流程

**Step 1 — 設定產品**

調整霜淇淋和聖代的售價與毛利率。毛利率用 slider 拖拉即可。

**Step 2 — 設定營運參數**

填入每月營業天數、每日時數、PT（兼職人員）的 cover 比例、時薪和人數。PT 排班成本會自動計算。

**Step 3 — 確認固定成本**

房租、水電、折舊、正職薪資等已有預設值，可依實際情況調整。

**Step 4 — 查看結果**

右側會即時顯示：
- BEP 月營業額（大數字）
- 不同產品組合下的銷量目標（月 / 日 / 時）
- 損益敏感度折線圖（綠色獲利、紅色虧損）
- 固定成本結構長條圖

**Step 5（選用）— 開啟酒場合作通路**

打開「酒場合作」開關，輸入 B2B 通路預估月售量，查看：
- 酒場返利 20% 後，愛嬌實際保留的毛利
- B2B 貢獻能抵銷多少固定成本
- 全通路（直營 + 酒場）的營業額拆解與綜合毛利率

## 注意事項

此 BEP 僅涵蓋固定成本（房租、薪資、折舊等），**尚未納入行銷推廣、促銷折扣、外送平台抽成等變動成本**。實際損益平衡點將高於此數字，建議加入變動成本後再作最終決策。
