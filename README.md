# Aikyo 冰店損益平衡分析

愛嬌冰店互動式 BEP（Break-Even Point）試算工具，適合與團隊溝通討論使用。

**Demo →** [Cloudflare Pages 部署連結]

---

## 功能

**左側參數（即時調整）**
- 產品設定：霜淇淋 / 聖代售價、各自毛利率
- 通路設定：酒場合作 B2B 通路，輸入月售數量，模擬返利後對固定成本的抵銷效果
- 營運設定：每月營業天數、每日時數、PT cover 比例、時薪、人數
- 固定成本：房租、水電、折舊、正職薪資、PT 勞保/勞退

**右側結果（自動更新）**
- BEP 月營業額大數字
- 銷量目標情境：全霜淇淋 / 各半 / 全聖代 / 自訂比例（slider）
- 三情境對比表：有效 GM、平均單價、月/日/時銷量目標
- 損益敏感度圖：綠色獲利、紅色虧損的 P&L 曲線
- 固定成本結構長條圖
- 酒場通路啟用時：全通路營業額拆解、綜合毛利率

---

## 注意事項

此 BEP 僅涵蓋固定成本（房租、薪資、折舊等），**尚未納入行銷推廣、促銷折扣、外送平台抽成等變動成本**。實際損益平衡點將高於此數字，建議加入變動成本後再作最終決策。
